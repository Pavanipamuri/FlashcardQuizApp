# Flashcard Quiz App

This is a simple quiz application built using Python and Flask.

## Features

* Add flashcards
* Take quizzes
* Simple and interactive interface

## Technologies Used

* Python
* Flask
* HTML
* CSS

## How to Run

1. Install Flask:
pip install flask
2. Run the application:
python app.py
3. Open the localhost link in your browser.
